## Getting Started[^1]

1. Clone the repo using the following command:

```console
$ git clone https://github.com/m-ikiara/c0nw4y.git
```

2. Navigate into the new directory and build the executable:

```console
$ make -B -j3
```
3. Run the executable and enjoy the show.

```console
$ ./c0nw4y
```
