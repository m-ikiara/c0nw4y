## Getting Started[^1]

1. Clone the repo using either of the following commands:

```console
$ git clone https://github.com/m-ikiara/c0nw4y.git
```

2. Navigate into the new directory and execute the following:

```console
$ make -B -j3
$ ./6utt3r
```
